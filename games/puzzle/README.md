# 拼拼中国（合集内置游戏）

本目录是「拼拼中国」中国地图拼图，作为合集 `kids-games` 的第三款游戏，
让合集自包含、双击 `file://` 即玩。纯前端、离线、零运行时依赖、`window.PP` 命名空间，
**与合集的 `core/`（`window.KG`）互不依赖、互不冲突**（多页结构，各页只加载自己的脚本与 CSS）。

## 来源（provenance）

最初来自独立项目 `pinpin-china`，现已合并入本合集、在本仓内统一维护；原独立仓库已归档为只读、不再单独演进。
合并时仅在首页 `index.html` 顶部加了一个 `← 合集` 返回链接，其余文件原样保留。

## 只含运行时

只保留游戏运行所需文件：`index.html`、`js/`、`css/styles.css`、`data/`（已预构建的省/地市 SVG 数据）。
未包含原项目的 `tools/`（`d3-geo` 投影、`puppeteer` e2e、源 GeoJSON）——那些只在「重新生成地图数据」时才需要，运行游戏用不到。
如需重新生成地图数据，应在补回相应数据工具链后再处理。
